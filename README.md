# Body Surface Area (BSA) Calculator for CAR-T Cell Lymphodepletion Dosing

## This Python program is designed to run on the TI-84 Plus CE Python Edition Calculator

### Current Release

* Version 4

### Description

This Python program accepts either Metric (cm & kg) or English (inches & pounds) values to calculate a patient's BSA, and then uses this calculated BSA to compute dosing levels typically used during the Lymphodepletion phase for Multiple Myeloma patients undergoing CAR-T Cell Therapy.

### Key Features

* Calculates both Du Bois BSA and Mosteller BSA.
* Uses Mosteller BSA to calculate daily and total dosing levels for Fludarabine and Cyclophosphamide.
