# Body Surface Area (BSA) Calculator for CAR-T Lymphodepletion Dosing

## This Python program is designed to run on the TI-84 Plus CE Python Edition Calculator

### Current Release

* Vesion 3

### Description

This Python program accepts either Metric (cm & kg) or English (inches & pounds) values to calculate a patient's BSA, and then uses this calculated BSA to compute typical dosing levels for CAR-T Lymphodepletion Dosing.

### Key Features

* Calculates both Du Bois BSA and Mosteller BSA.
* Uses Mosteller BSA to calculate daily and total dosing levels for Fludarabine and Cyclophosphamide.
